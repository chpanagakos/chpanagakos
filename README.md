## What I'm building

**Misconception diagnosis for Greek physics exams** — [live demo](https://huggingface.co/spaces/chpanagakos/greek-physics-rag) · [repo](https://github.com/chpanagakos/greek-physics-rag)

A RAG pipeline that takes a Greek national university-entrance examinations (Panhellenic exams)
physics problem and a student's incorrect attempt, grounds a diagnosis in retrieved curriculum material, and surfaces candidate misconceptions from a closed, tutor-audited taxonomy for a tutor to confirm. Frontier models can often *solve* these problems without retrieval — the point here is grounded, auditable diagnosis: every claim cites the corpus chunk it rests on, and a retrieval on/off toggle shows the difference between a diagnosis a tutor can check and one they have to trust.
